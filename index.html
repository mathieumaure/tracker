Comment obtenir un token GitHub ?
  1. Ouvrez github.com → Settings → Developer settings → Personal access tokens → Tokens (classic).
  2. « Generate new token (classic) », cochez uniquement la case gist.
  3. Copiez le token (ghp_…) et collez-le ci-dessus.
  4. Sur le 1er appareil : laissez « Identifiant du Gist » vide. L'app vous donnera l'ID à reporter sur le 2e appareil.
